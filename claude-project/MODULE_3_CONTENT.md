# MODULE 3 — CONTENT PAGE BUILD
# Use: When creating or improving a specific page.
# Trigger: "Run Module 3. Page: [page name]. Target: [who it is for or keyword]."

---

## WHAT THIS MODULE DOES

Builds or rewrites one page completely.
Every page is written for a real human first, then optimized for search and AI.
Each page serves ONE primary intent only.

---

## EXAMPLE TRIGGERS

- "Run Module 3. Page: What to Expect. Target: first-time church visitor Pretoria."
- "Run Module 3. Page: Donate. Target: donor considering supporting an NPO."
- "Run Module 3. Page: Prayer. Target: person searching 'prayer for anxiety'."
- "Run Module 3. Page: About. Target: new visitor wanting to trust FGI."
- "Run Module 3. Page: Youth. Target: parent searching 'youth mentorship Pretoria North'."

---

## TWO VISITOR PATHS — CHOOSE ONE PER PAGE

**Path A — Seeker / Visitor**
Person needs: hope, healing, belonging, prayer, practical community help.
Write with: warmth, clarity, real answers, no church jargon.

**Path B — Donor / Supporter**
Person needs: trust, proof of impact, legitimacy, stewardship confidence.
Write with: facts, NPO proof, transparency, clear donation path.

---

## REQUIRED OUTPUT — IN THIS ORDER

**1. PAGE INTENT**
One sentence stating exactly what this page is for and who it serves.

**2. PAGE STRUCTURE**
- H1 (primary keyword + human clarity)
- H2s (logical sections)
- Key facts section (address, times, contact — in text, not images)
- Call to action placement

**3. FULL PAGE COPY**
- Written for the target visitor
- Optimized for the target search term
- Answer-first: key facts in the first 100 words
- Entity consistent: "Fountain of Grace International" or "FGI" — no variations
- Plain language — no church jargon for seekers, no vagueness for donors
- Ends with one clear next action

**4. META TITLE**
Under 60 characters. Keyword-rich. Human-readable.

**5. META DESCRIPTION**
Under 160 characters. Compelling. Includes target keyword naturally.

**6. OPEN GRAPH TAGS**
- og:title
- og:description
- og:image (note: image must be 1200x630px)
- og:url

**7. JSON-LD SCHEMA**
Correct schema type for this page. Validated. No fake data.

**8. INTERNAL LINKS**
- 2–3 pages this page should link to
- Which existing pages should link to this new page

**9. LLMS.TXT UPDATE**
Should this page be added or updated in /llms.txt? Yes or No. If yes, provide the line.

**10. CONTENT DECAY NOTE**
Does this page contain dates, times, events, or prices that will go stale?
If yes, flag it for review and note when to check it.

---

## CONTENT STANDARDS

- No invented testimonials, statistics, or impact numbers
- All facts must match verified brand facts in Project Instructions
- POPIA compliant if the page contains a form
- Form must include a one-line data disclosure near the submit button
- WhatsApp CTA uses wa.me/27XXXXXXXXX format (operator fills correct number)
